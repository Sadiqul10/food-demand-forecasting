# 🍽️ Food Demand Forecasting

A Machine Learning-based Food Demand Forecasting Web Application developed using Python and Streamlit. This project helps analyze historical food demand data, visualize sales trends, and predict future food demand using forecasting techniques.

---

## 🌐 Live Demo

🔗 [https://food-demand-forecasting.streamlit.app/](https://food-demand-forecasting.streamlit.app/)

---

## 📌 Project Overview

Food Demand Forecasting is an intelligent forecasting system designed to predict future food demand based on historical data. The application uses data preprocessing, visualization, and machine learning models to provide accurate demand forecasting results.

The project is useful for:

* Restaurants
* Food delivery businesses
* Inventory management
* Supply chain optimization
* Retail food analytics

## ⚙️ Installation Guide

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/food-demand-forecasting.git
```

---

### Step 2: Navigate to Project Folder

```bash
cd food-demand-forecasting
```

---

### Step 3: Create Virtual Environment

```bash
python -m venv venv
```

---

### Step 4: Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

---

### Step 5: Install Required Packages

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Application will run at:

```bash
http://localhost:8501
```
